# SEC Rules 605 and 606 - Execution Quality and Routing Disclosure

Market transparency rules requiring disclosure of execution quality (605) and order routing practices (606).

## Rule 605 - Execution Quality Disclosure

### Purpose

Enable comparison of execution quality across market centers. Helps brokers fulfill best execution obligations.

### Who Must Report

Market centers that execute orders:
- Exchanges
- ATSs executing 5%+ of volume in any NMS stock
- Market makers

### Reporting Frequency

Monthly reports, published within one month of month-end.

### Required Statistics

**By order type and size bucket:**

| Metric | Definition |
|--------|------------|
| Covered orders | Market and marketable limit orders |
| Shares executed | Total shares filled |
| Shares cancelled | Total shares cancelled |
| Effective spread | 2 x |execution price - midpoint| |
| Effective/quoted ratio | Effective spread / quoted spread |
| Price improvement | Orders receiving better than NBBO |
| Fill rate | Percent of shares executed |
| Time to execution | Speed buckets |

### Size Buckets (Pre-2024)

| Bucket | Share Range |
|--------|-------------|
| 1 | 100-499 |
| 2 | 500-1999 |
| 3 | 2000-4999 |
| 4 | 5000-9999 |
| 5 | 10000+ |

### 2024 Modernization

SEC adopted significant updates effective 2025-2026:

**Expanded coverage:**
- Odd-lot orders (< 100 shares)
- Fractional shares
- All NMS stocks (not just covered)

**New metrics:**
- E/Q ratio at time of order receipt
- Size improvement
- Midpoint vs quoted execution
- Fill rate by notional value

**New size buckets:**
- More granular (e.g., 1-99, 100-499, etc.)
- Notional-based buckets

**Short sale indicator:**
- Separate statistics for short sale orders

### Using 605 Data

**For broker best execution:**
- Compare venue execution quality
- Document venue selection rationale
- Identify systematic deficiencies

**For research:**
- Analyze spread components across venues
- Study execution quality trends
- Compare internalization vs exchange

**Limitations:**
- Aggregated data (no order-level)
- Self-reported by market centers
- Historical lag (monthly)

## Rule 606 - Order Routing Disclosure

### Purpose

Transparency into broker routing decisions and payment for order flow arrangements.

### Who Must Report

Broker-dealers routing non-directed customer orders.

### Reporting Types

**Quarterly aggregated report (public):**
- Top 10 venues for each order type
- PFOF and profit-sharing arrangements
- Material relationships description

**Customer-specific report (on request):**
- Venue routing for that customer's orders
- More detailed breakdown

### Required Disclosure

**For each venue:**

| Item | Description |
|------|-------------|
| Venue name | Exchange, ATS, or market maker |
| % of non-directed orders | Share routed to venue |
| % of market orders | Subset for market orders |
| % of limit orders | Subset for limit orders |
| Net payment received | PFOF amount (or paid) |
| Material relationships | Ownership, agreements |

### Order Categories

| Category | Definition |
|----------|------------|
| S&P 500 stocks | Large cap equities |
| Non-S&P 500 stocks | Other equities |
| Options | Listed options |

### PFOF Disclosure

Must disclose:
- Total payment received per venue
- Average payment per share
- Description of payment arrangement

### 606(b)(3) - Held NMS Stocks

Enhanced disclosure for held orders:
- More detail on execution quality at each venue
- Comparison metrics

### Using 606 Data

**For retail investors:**
- Understand where orders go
- Evaluate PFOF arrangements
- Compare broker routing

**For research:**
- Study broker routing patterns
- Analyze PFOF economics
- Track routing changes over time

**For compliance:**
- Document routing relationships
- Support best execution review
- Conflict of interest management

## Quant Applications

### Execution Quality Research

Using 605 data:
```
1. Collect monthly 605 reports from venues of interest
2. Aggregate by security tier and time
3. Calculate effective spread trends
4. Compare price improvement rates
5. Analyze fill rate patterns
```

### Routing Analysis

Using 606 data:
```
1. Collect quarterly 606 from brokers
2. Map routing percentages by venue type
3. Calculate PFOF per share
4. Track routing changes over time
5. Correlate with execution quality
```

### Limitations to Note

**605 limitations:**
- Aggregation masks order-level variation
- Size bucket granularity
- Timing (end of month + lag)

**606 limitations:**
- Quarterly frequency
- Aggregated by category
- Non-directed orders only

### Academic Research Using 605/606

Studies have used these disclosures to analyze:
- PFOF impact on execution quality
- Venue competition effects
- Retail vs institutional execution
- Internalization economics
